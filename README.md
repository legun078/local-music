# local-music

iPhone에 저장된 MP3·M4A를 재생·정리하는 로컬 전용 iOS 음악 플레이어입니다.

- 스트리밍·서버 연동 없음
- 파일 앱에서 음악 가져오기 / 폴더 연결
- 앨범·아티스트·재생목록 보관함
- 싱크 가사, ReplayGain 자동 볼륨 조절

## 문의

- 이메일: **021gundam@gmail.com**
- 버그·기능 요청: GitHub Issues 사용 가능

## 요구 사항

- iOS 17.0 이상
- MP3, M4A 등 로컬 오디오 파일

## 개인정보 처리방침 (Privacy Policy)

Local Music은 사용자의 음악 파일을 **기기 안에서만** 처리합니다.

- 계정 가입·로그인 없음
- 음악 파일·메타데이터를 개발자 서버로 전송하지 않음
- 광고·추적·분석 SDK 미사용
- 수집하는 개인정보 없음
